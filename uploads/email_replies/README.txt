DROP EMAIL REPLIES HERE
========================

Drop any email (.txt, .eml, .pdf, or .docx) from a prospect or partner that you want Claude to draft a response to.

The Inbox Zero Assistant will pick these up automatically on the next run (9am, 3pm, or manual trigger) and generate a draft reply in Gmail.

Tips:
- Copy-paste the email body into a .txt file if you don't have the raw .eml
- Name the file something descriptive: e.g. "reply_from_victor_gonzalez.txt"
- Files are NOT deleted after processing — remove them manually once you've sent the reply
